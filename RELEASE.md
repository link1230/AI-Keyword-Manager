# PromptManager v0.1.0 — 首个正式版

> AI 提示词管理器 — Windows 原生桌面应用。组织、搜索、复用你的 AI 提示词模板。

---

### 下载 / Download

[PromptManager_Setup.exe](https://github.com/你的用户名/prompt-manager/releases/download/v0.1.0/PromptManager_Setup.exe)（4.5 MB）

### 功能亮点 / Highlights

- 分类树组织提示词，支持新增 / 改名 / 删除
- 标签 + AI 模型双筛选（ChatGPT / Claude / Gemini / DeepSeek / Kimi…）
- `Ctrl+K` 全局快速搜索，键盘导航
- `{变量名}` 占位符，中文支持，复制前一键填充，默认值自动记忆
- 粘贴 / 拖拽 / 按钮插入图片，base64 存储，JSON 导入导出完整保留
- 中英双语界面一键切换
- 深色 / 浅色主题（暖奶油 editorial + OLED 深色）
- IndexedDB 本地存储，无容量限制，零云依赖
- 启动封面动画

### 安装 / Install

双击 `PromptManager_Setup.exe` 安装即可。  
Windows 10/11 自带运行环境，无需额外安装。
